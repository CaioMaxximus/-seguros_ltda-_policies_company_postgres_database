# SEGUROS LTDA
## PostgreSQL Database for Insurance Services

This project focuses on designing and building a PostgreSQL database for an insurance company.

It covers the creation of a structured schema to manage core business entities such as customers, policies, payments, claims, and agents.

The goal is to study and apply database design principles, including normalization, relationships, constraints, and query optimization, in a real-world scenario.
